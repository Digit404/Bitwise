Welcome to DFJK

DFJK is a typing game where you have to type the letters on the screen as fast as possible without making mistakes. The game is designed to be played with the titular keys: D, F, J, and K. The game will start when you press the first key.

Press Space or Enter to generate a new chart, and press r to restart.

In advanced mode, chords appear which have multiple keys you have to press at the same time to advance. Press Space to press all the keys at once.
